# TODO APP Built with 

## Task:
### Create a full-stack To-Do application. 

## Features:
### - Display a list of To-Do items.
### - Add a new To-Do item.
### - Edit an existing To-Do item.
### - Delete a To-Do item.
### - Mark a To-Do item as completed.

## Project Setup:
### - clone:git@github.com:Codehouze/TODO_FE.git
### - cd TODO_FE
### - yarn install
